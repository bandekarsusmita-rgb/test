# Susmita Bandekar Portfolio

A polished static portfolio site ready for GitHub Pages.

## Included files
- `index.html`
- `styles.css`
- `Susmita_Bandekar_Resume.pdf`

## Fastest way to publish on GitHub for free

### Option 1: Upload in the browser
1. Sign in to GitHub.
2. Create a new **public** repository, for example `susmita-portfolio`.
3. Click **Add file** → **Upload files**.
4. Upload `index.html`, `styles.css`, and `Susmita_Bandekar_Resume.pdf`.
5. Commit the files to the `main` branch.
6. Open **Settings** → **Pages**.
7. Under **Build and deployment**, choose **Deploy from a branch**.
8. Select branch **main** and folder **/(root)**.
9. Save.
10. Wait 1–3 minutes and open:
   `https://YOUR-USERNAME.github.io/susmita-portfolio/`

### Option 2: Exact git commands
Run these commands in Terminal after creating an empty public repo on GitHub:

```bash
git clone https://github.com/YOUR-USERNAME/susmita-portfolio.git
cd susmita-portfolio
cp /path/to/index.html .
cp /path/to/styles.css .
cp /path/to/Susmita_Bandekar_Resume.pdf .
git add .
git commit -m "Add portfolio site"
git push origin main
```

Then go to **Settings** → **Pages** and select **main** + **/(root)**.
