# Susmita Bandekar Portfolio

This is a simple static portfolio site built from resume content.

## Files
- `index.html`
- `styles.css`

## Publish free with GitHub Pages
1. Create a GitHub account and sign in.
2. Create a new public repository, for example `susmita-portfolio`.
3. Upload `index.html` and `styles.css` to the repository root.
4. Go to **Settings** → **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select **main** branch and **/(root)** folder.
7. Save.
8. Wait 1–3 minutes. Your portfolio will appear at:
   `https://YOUR-USERNAME.github.io/susmita-portfolio/`

## Optional customizations
- Add your professional photo
- Add a downloadable PDF resume
- Add case studies or project pages
- Add a custom domain later if needed
